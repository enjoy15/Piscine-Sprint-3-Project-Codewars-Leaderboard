# Codewars Leaderboard

A small web app that compares Codewars users by overall and language-specific rank scores.

## Features
- Enter comma-separated usernames and fetch live data from the Codewars API.
- Automatically discover all available language rankings across users.
- Switch between overall and language-specific leaderboards.
- Show username, clan, and score sorted highest to lowest.
- Hide users who have no rank in the selected language.
- Highlight the top-ranked user.
- Show helpful errors for invalid users and failed API requests.

## Run locally
1. Install dependencies:
   npm install
2. Run tests:
   npm test
3. Open `index.html` in your browser.

## Deployment
This project is configured for Netlify automatic deployment. Changes pushed to the `main` branch are automatically deployed.
